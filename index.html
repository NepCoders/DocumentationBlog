Setup - Installing Theme
You will need a domain and Wordpress installed in your hosting account first.

Like any other Wordpress themes, these are prerequisities to using our themes.

Once you have Wordpress ready, please log into the admin section and then go to Appearance > Themes and click the blue "Add New" button on top.



In the next section, click on Upload Theme and then navigate to where you have the theme on your computer by choosing "Choose File" button and then select the zipped theme folder and then clcik on "Install Now" Button.

Options Panel - Layout
(screenshot)
Here you have all the options for making changes to single page page and single post page.


1) Show Author: If enabled, the name of the author will be shown in the single page.

2) Show Date: The date the post was publshed will show if the optiion is activated.

3) Show Category: The category the post is included in will be displayed on the post if enabled.

4) Show Comments: The comment form and all the comments made by vistors, if any, will be showm if this option is enabled
